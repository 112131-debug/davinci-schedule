# 達文西機械手臂排刀預約系統 — UI 設計簡報

## 專案背景

北醫體系三家醫院（附醫、萬芳、雙和）各配置達文西機械手臂手術設備，目前排刀資訊僅以純文字描述呈現，缺乏視覺化管理工具。本系統目的為提供直觀的週曆式介面，讓排刀人員清楚掌握兩台設備的使用現況，並可線上填寫預約。

---

## 設計目標

- 醫療專業感：乾淨、簡潔、高對比，適合臨床場域使用
- 資訊一目瞭然：在一個畫面內同時呈現兩台設備的整週排程
- 操作直覺：點擊空白時段即可進入預約流程

## 元件說明

### 1. Header（頂部導覽列）

- 左側：系統名稱「達文西排刀預約管理系統」＋北醫體系 Logo（深藍色底，白字）
- 右側：三個 Tab 按鈕，切換分院（附醫 / 萬芳 / 雙和），選中院別以藍色底色高亮
- 高度：64px，陰影分隔主內容

---

### 2. Toolbar（操作列）

- 週期導覽：左右箭頭切換週次，中間顯示「YYYY年M月D日 — M月D日」
- 快速按鈕：
  - `今日`：跳回當週
  - `本月預覽`：切換至月曆縮覽模式（簡化版，僅顯示是否有排刀）
  - `歷史紀錄`：切換至過去排刀查詢模式（可依科別、機台、日期範圍篩選）
  - `匯出`：下載當週排程 PDF 或 Excel

---

### 3. 主週曆格（核心元件）

#### 欄位結構
- 最左欄：時間軸，07:00 — 18:00，每格代表 1 小時，行高 52px
- 其餘每天分為兩個子欄：`機台一` 與 `機台二`，以細線分隔
- 每日欄寬相等，日期標頭標示「週X（M/D）」格式
- 若為今日，日期標頭以藍色圓圈標示

#### 預約區塊（Booking Block）
- 已預約時段：以科別色塊填滿，圓角矩形，高度依時長動態延伸
- 色塊內顯示（由上至下）：
  - 科別名稱（粗體，12px）
  - 主刀醫師（11px）
  - 手術名稱（11px，超出截斷加 ...）
- 色塊右上角：小型鉛筆 icon（hover 才顯示），點擊可編輯
- 空白時段：淺灰色背景，hover 時顯示「＋ 點擊預約」提示

#### 午休視覺提示
- 12:00–13:00 的列以斜線淡灰填色，標示「午休」，但仍可預約（特殊情況）

---

### 4. 預約表單（Modal 彈窗）

點擊空白時段或「新增預約」按鈕後，從右側滑入或以置中 Modal 呈現。

**欄位清單：**

| 欄位名稱 | 類型 | 說明 |
|---------|------|------|
| 機台 | 單選（機台一 / 機台二） | 預設帶入點擊的機台 |
| 日期 | 日期選擇器 | 預設帶入點擊的日期 |
| 開始時間 | 時間選擇器（整點） | 預設帶入點擊的時段 |
| 結束時間 | 時間選擇器（整點） | 依預估時長自動計算 |
| 科別 | 下拉選單 | 泌尿科 / 一般外科 / 婦產科 / 胸腔外科 / 其他 |
| 主刀醫師 | 文字輸入 / 下拉 | 可輸入或從名單選取 |
| 助理醫師 | 文字輸入（可選） | |
| 手術名稱 | 文字輸入 | |
| 預估手術時長 | 數字輸入（小時） | 同步更新結束時間 |
| 病人識別碼 | 文字輸入（選填） | 住院號或病歷號 |
| 備註 | 多行文字（選填） | |

**操作按鈕：**
- `確認預約`（主要按鈕，深藍色）
- `取消`（次要按鈕，灰色框線）
- 若為編輯模式，額外顯示 `刪除此預約`（紅色文字按鈕）

---

### 5. 歷史紀錄模式

切換至歷史紀錄後，版面調整為：

- 上方：篩選列（日期範圍、科別、機台、醫師姓名）
- 中間：週曆呈現過去資料，色塊以半透明處理，右側標示「已完成」
- 下方：統計摘要卡片
  - 所選期間機台一使用率 %
  - 所選期間機台二使用率 %
  - 最高使用科別
  - 排刀次數總計

## 視覺風格指引

- **整體風格**：醫療科技感，乾淨俐落，白底為主，使用深藍（#1E3A5F）作為品牌主色
- **字型**：無襯線字體（建議 Noto Sans TC 或 Inter + 中文字型）
- **圓角**：卡片 12px，按鈕 6px，色塊 6px
- **陰影**：僅用於 Modal 彈窗與卡片（輕陰影，避免過重）
- **間距**：格線寬鬆，保留足夠留白，不壓縮資訊
- **響應式**：設計以 1440px Desktop 為主，不需考慮手機版

## 產出需求說明（給 AI 設計師）

請根據以上設計簡報，產出以下畫面的高保真 UI 設計圖：

1. **主畫面**：完整週曆，含示意資料填入、科別色塊、機台一/二並列
2. **預約表單**：Modal 彈窗開啟狀態，含欄位填寫示意
